Gesture based tool for sterile browsing of radiology images (Team ID: PNT2022TMID06155)

Instructions

  • This tool is used for sterile browsing of radiology images.
  • To use this app first click the start link on the top.
  • Then upload the radiology images.
  • Then start browsing the radiology images.
  • The gestures are:
    • 0 to start browsing(Before every gesture we have to start the browsing first).
    • 1 to rotate the image clockwise.
    • 2 to rotate the image couter-clockwise.
    • 3 for next image.
    • 4 for previous image.
    • 5 to reset the rotation for current image
